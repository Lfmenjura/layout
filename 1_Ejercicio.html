CONTENEDORES

CONTENEDOR

Use la .container clase para crear un contenedor receptivo de ancho fijo,Tenga en cuenta que su ancho ( max-width) cambiará en diferentes tamaños de pantalla

CONTENEDOR FLUIDO

Use la .container-fluidclase para crear un contenedor de ancho completo, que siempre abarcará todo el ancho de la pantalla ( widthsiempre 100%)


CONTENEDOR RECEPTIVO

También puede usar las .container-sm|md|lg|xlclases para crear contenedores receptivos. El max-widthdel contenedor cambiará en diferentes tamaños de pantalla / ventanas gráficas
